# Standard Operating Procedures

This directory contains SOPs — reusable step-by-step procedures that agents follow for recurring tasks.

## Structure

Create one `.md` file per domain or procedure type. Example:

```
sops/
├── README.md           (this file)
├── content-drafting.md
├── research.md
└── publishing.md
```

## Format

Each SOP should include:
1. **Purpose** — what problem this solves
2. **Trigger** — when to use this SOP
3. **Steps** — numbered, actionable steps
4. **Notes** — edge cases, gotchas

Agents will read these files before starting work in the relevant domain.
